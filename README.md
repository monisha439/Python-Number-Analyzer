# Python Number Analyzer

## Description
A menu-driven Python application that performs various number operations.

## Features
- Armstrong Number
- Factorial
- Fibonacci Series
- Neon Number
- Odd/Even
- Palindrome Number
- Perfect Number
- Prime Number
- Reverse Number
- Strong Number

## Technologies Used
- Python 3

## Project Structure

```
Main.py
Armstrong.py
Factorial.py
Fibonacci.py
Neon.py
Oddeven.py
Palindrome.py
Perfect.py
Prime.py
Reverse.py
Strong.py
README.md
```

## How to Run

1. Open the project folder.
2. Run `Main.py`.
3. Choose an option from the menu.
