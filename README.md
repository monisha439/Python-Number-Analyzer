# Python Number Analyzer

## 📌 Project Description

Python Number Analyzer is a menu-driven console application developed using Python. It allows users to perform various mathematical and number-related operations through a simple menu interface.

---

## 🚀 Features

- Armstrong Number
- Factorial
- Fibonacci Series
- Neon Number
- Odd or Even Number
- Palindrome Number
- Perfect Number
- Prime Number
- Reverse Number
- Strong Number

---

## 🛠️ Technologies Used

- Python 3
- Functions
- Modules
- Loops
- Conditional Statements

## ▶️ How to Run

1. Clone or download this repository.
2. Open the project in VS Code.
3. Open Terminal.
4. Run the following command:

```bash
python Main.py
```
